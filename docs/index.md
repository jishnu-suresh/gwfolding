# gwfolding

```{image} ../assets/logo.png
:alt: gwfolding logo
:width: 360px
:align: center
```

**gwfolding** folds *Stochastic Intermediate Data (SID)* frames to one sidereal
day for stochastic gravitational-wave searches. The output can be written as
folded SID (`.gwf`) frames.

Mathematical details are described in the LIGO technical document **T0900093**.
